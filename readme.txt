Chrome folder contains the files necessary to create a chrome extension, the files in the chrome folder are written in HTML,JSON and JS

WebServiceDemo folder contains the files necessary to deploy a webservice which is used to insert the data into table,webservice is written in spring framework.

Chrome extension will extract the 10 digit numbers from the current webpage and forms a comma seperated list which is then sent to webservice using post method and webservice will write the data into database
